Dem Smart falls under the Smart Home track outlined by Boston Hacks. It is specifically designed for use with people suffering from dementia. Reminders can be automated for specific times using the Bose SoundTouch 10, all depending on the various needs of our user.

Project Info


Challenge

Learning how to utilize IoT devices to realize the dream of the smart home:
-IoT Device in Question: Bose SoundTouch 10
-Device automatically notifies the user of pre-determined tasks

Audience

Studying the intimate interaction of dementia patients and technology:
-Determine how technology can better their lives
-Review what technology currently exists in the market

Finance

$341,840 is the estimated lifetime cost of care for an individual living with dementia
-An overwhelming cost for many families/patients affected by dementia
-Our product help minimize this cost

Bose SoundTouch 10

-IoT component used to notify the user -Integrated into a home easily

Bose SoundTouch API

-Audio Notification API allows for the notifications to be sent to the user

Google Calendar API

-Sets the specific events for the user and allows for easy modifications
